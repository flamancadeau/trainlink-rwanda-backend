# TrainLink Rwanda - Backend API

This is the backend service for the **TrainLink Rwanda** application, which provides various functionalities like train management, user authentication, image uploads, and more. This API is built with **Node.js**, **TypeScript**, **Express**, **MongoDB**, and uses several additional services like **Cloudinary** for image uploads and **Nodemailer** for email notifications.

## Installation

To get the project up and running on your local machine, follow the steps below:

### Prerequisites

- **Node.js** (v16.x or higher) - [Download Node.js](https://nodejs.org/en/download/)
- **MongoDB** (Local or cloud instance via MongoDB Atlas)
- **Cloudinary Account** (For image uploads)
- **Gmail (or any email service) Account** (For sending emails)
